HTML არის ტიმ ბერნერს-ლის მიერ 1993 წელს შექმნილი აღწერის ენა, რომელიც განკუთვნილია ვებ-გვერდების და საიტების შესაქმნელად და ინფორმაციის გასავრცელებლად ინტერნეტის საშუალებით.

CSS არის სტილთა ფურცლების ენა, რომელიც გამოიყენება მონიშვნის ენით დაწერილი დოკუმენტის წარდგინების აღსაწერად, მაგალითად HTML-ის გამოყენებით.[1] CSS არის მსოფლიო აბლაბუდის ტექნოლოგიური ქვაკუთხედი, HTML-თან და JavaScript-თან ერთად.

JAVA SCRIPT არის პროგრამირების ერთ-ერთი ფართოდ გავრცელებული ენაა. სახელწოდება JavaScript, როგორც სავაჭრო ნიშანი დარეგისტრირებულია ფირმა Sun Microsystems, Inc-ის მიერ.
